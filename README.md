# vue-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
In order to deploy to the webserver, su -c "cp -r /dist/* /var/www/html"

### Lints and fixes files
```
npm run lint
```
